# Impulso Digital

## Descrição
Impulso Digital é um site para ajudar microempresas a crescerem no ambiente digital. Oferece estratégias inteligentes, automação e presença online forte para pequenos empreendedores.

## Funcionalidades
- Página inicial com apresentação da empresa
- Seção "Sobre Nós" explicando a missão
- Contato com informações de telefone, email e WhatsApp
- Formulário de solicitação de orçamento que redireciona para Google Forms
- Design responsivo com gradiente azul escuro e elementos visuais

## Tecnologias Utilizadas
- HTML5
- CSS3 (com gradientes e responsividade)
- JavaScript (para manipulação do formulário)
- Font Awesome (para ícones)

## Como Executar
1. Clone o repositório:
   ```
   git clone https://github.com/Pyjhonatan/projeto-No-Code-impulso-digital-.git
   ```
2. Abra o arquivo `impulso.html` em um navegador web.

## Estrutura do Projeto
- `impulso.html`: Arquivo principal HTML
- `impulso.css`: Estilos CSS
- `impulso.js`: Script JavaScript
- `images/`: Pasta com imagens (ex: logo.png)
